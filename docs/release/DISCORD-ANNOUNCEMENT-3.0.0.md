# VOID-CRAFT Launcher 3.0.0

## Discord Announcement

```md
## VOID-CRAFT Launcher 3.0.0 je venku

Po delší a pořádně velké přestavbě pouštíme ven **Launcher 3.0.0**.
Tohle není jen další malý update, ale **kompletní redesign launcheru** skoro od základů.

### Co je nové

- **Nový shell a dashboard**
  Launcher má nový layout, přehlednější navigaci, produkční dashboard, context dock a plynulejší přechody mezi stránkami.

- **Server Hub**
  Centrální místo pro VOID-CRAFT server, community servery a rychlé připojení s lepším feedbackem při spouštění.

- **Achievement Hub**
  Sezónní progress, leaderboard feel a čistší rozdělení achievementů podle reálného významu.

- **Skin Studio**
  Lepší navázání na identitu hráče, UUID a historii skinů.

- **Instance Workspace a Creator Studio**
  Detail instance už není jen statická stránka. Je z toho reálný workspace pro obsah, export/import, snapshots a úpravy souborů.

- **Themes, localization a polished UX vrstva**
  Více motivů, runtime CZ/EN lokalizace, loading skeletony, empty states, lepší feedback a čistější design system.

- **Nová FUTURE stránka**
  Přímo v launcheru je nová roadmap surface, která se načítá z GitHub `future.md` a ukazuje, kam se launcher může posouvat dál.

### Co se zlepšilo technicky

- lepší architektura launcheru a rozdělení logiky
- robustnější content/news feed s cache a fallback chováním
- čistější build pipeline
- znovu ověřené `Debug` i `Release` buildy před vydáním

### Proč je to důležité

Cílem bylo, aby launcher nepůsobil jako vedlejší utilita, ale jako **plnohodnotný vstupní portál do VOID-CRAFT ekosystému**.
Nejen tlačítko `Hrát`, ale místo pro progres, obsah, community signály a další směr produktu.

### Co teď

Pokud launcher už máte, doporučuju přejít na **3.0.0**.
Pokud ho budete procházet detailněji, klidně pište feedback na UI, flow i to, co chcete vidět v dalších verzích.

Díky všem, kdo to testovali, rozbíjeli a tlačili kvalitu výš.
Tohle je konečně verze, kterou se nebojím pustit veřejně ven.
```
