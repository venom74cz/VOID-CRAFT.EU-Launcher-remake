# Přispívání

Tento repozitář není otevřený pro nevyžádané příspěvky do kódu ani další
použití.

Než otevřeš pull request, pošleš kód nebo použiješ jakoukoli část launcheru
mimo schválenou VOID-CRAFT práci, nejdřív kontaktuj správce repa a domluvte si
rozsah, oprávnění a podmínky spolupráce.

Bez předchozího schválení jsou vítané:

- bug reporty
- reprodukovatelné hlášení problémů
- UX feedback
- návrhy funkcí

Bez předchozí písemné dohody neposílej:

- pull requesty s kódem
- redistribuované fork verze nebo rebrandy
- odvozené launchery nebo znovu použité moduly
- zabalené buildy určené pro veřejné nebo soukromé nasazení

Pokud bude spolupráce schválena dopředu, autorství a licenční podmínky by měly
být domluvené ještě před mergem.
